# Говорун
Библиотека аудиокниг.  
Поддерживаются аудиокниги в форматах M4B/M4A/MP3.  
Аудиокнига должна состоять только из одного файла.

## Технологии
- Интерфейс: WPF
- Платформа: NET 8.0
- База данных: LiteDB

## Пакеты NuGet верхнего уровня
- LiteDB 5.0.20
- MahApps.Metro.IconPacks 5.0.0
- z440.atl.core 5.25.0

## Библиотеки
- Gemiyur 2.0.0
